## 🔄 Алгоритм работы

```
┌──────────────┐
│ Игрок        │
│ нажимает 'C' │
└──────┬───────┘
       │
       ▼
┌──────────────────────┐
│ Открывается RGB      │
│ колесо               │
└──────┬───────────────┘
       │
       ▼
┌──────────────────────┐
│ Игрок выбирает цвет: │
│ 1. Оттенок (колесо)  │
│ 2. Насыщенность      │
│ 3. Яркость           │
└──────┬───────────────┘
       │
       ▼
┌──────────────────────────────┐
│ Мод анализирует:             │
│ • R (Red) = 0-255            │
│ • G (Green) = 0-255          │
│ • B (Blue) = 0-255           │
└──────┬───────────────────────┘
       │
       ▼
┌───────────────────────────────┐
│ Сравнивает с базой блоков:    │
│ √[(R₁-R₂)² + (G₁-G₂)² +      │
│    (B₁-B₂)²]                  │
└──────┬────────────────────────┘
       │
       ▼
┌──────────────────────────────┐
│ Сортирует по близости        │
│ (от меньшей разницы          │
│  к большей)                  │
└──────┬───────────────────────┘
       │
       ▼
┌──────────────────────────────┐
│ Показывает ТОП-10            │
│ самых похожих блоков         │
└──────────────────────────────┘
```

## 🚀 Расширяемость

Легко добавить новые блоки:

```java
blocks.add(new BlockColor("Название", new Color(R, G, B)));
```

Где:
- `R` = Red (0-255)
- `G` = Green (0-255)
- `B` = Blue (0-255)
